### 何时使用

用于反馈状态的组件，向用户明确传达系统正在处理中的等待状态。

### 组件常见用法

带文字的加载组件，icon 有循环动画。建议 icon 在左，文字在右。例如用在用户发送消息后。减少用户长时间等待的焦虑。
